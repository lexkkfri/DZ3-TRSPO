# Паралельні обчислення: Гіпотеза Колаца

Програма реалізує багатопотоковий алгоритм обчислення кроків гіпотези Колаца (проблема $3n+1$) для 10 000 000 натуральних чисел.

### Опис завдання
Метою є демонстрація підходу еволюції даних у паралельних обчисленнях. Оскільки кількість кроків для кожного числа неможливо передбачити математично, використовується динамічний розподіл задач для ефективного завантаження процесора.

### Технічна реалізація
* **Мова:** Java.
* **Інструменти:** `ExecutorService` (FixedThreadPool).
* **Алгоритм:** Головний потік розбиває діапазон чисел на пакети та передає їх у пул потоків. Це дозволяє уникнути простоїв при нерівномірній складності обчислень.

### Результат роботи
Програма розраховує та виводить у консоль:
1.  Загальну кількість кроків для всіх чисел.
2.  Середню кількість кроків (Average Stopping Time).
3.  Час виконання програми.
